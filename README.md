# IoT Smart Anganwadi System

An IoT-based Smart Anganwadi monitoring system for real-time ICDS tracking and malnutrition risk prediction.

## Features
- Real-time child health monitoring
- Malnutrition risk prediction
- IoT sensor integration
- Cloud monitoring using Blynk
- Growth tracking & alerts

## Technologies Used
- ESP32
- IoT Sensors
- Blynk
- Embedded C / Arduino IDE

## Project Files
- PPT Presentation
- Project Report
- Abstract
- System Architecture

## Objective
To improve Anganwadi child monitoring through IoT-based real-time health tracking and early malnutrition detection.

## Future Scope
- AI-based prediction
- Mobile app integration
- Cloud database
- SMS alert system
